# Простая инструкция: Установка Git

## Способ 1: Скачать установщик (Самый простой) ⭐

1. **Откройте браузер** и перейдите по ссылке:
   ```
   https://git-scm.com/download/win
   ```

2. **Скачайте установщик** - нажмите на большую кнопку "Download for Windows"

3. **Запустите скачанный файл** (например, `Git-2.xx.x-64-bit.exe`)

4. **В установщике:**
   - Нажимайте "Next" на всех экранах
   - Можно оставить все настройки по умолчанию
   - Нажмите "Install"
   - Дождитесь завершения установки
   - Нажмите "Finish"

5. **ВАЖНО: Перезапустите PowerShell или терминал!**
   - Закройте все окна PowerShell/CMD
   - Откройте новое окно PowerShell

6. **Проверьте установку:**
   ```powershell
   git --version
   ```
   
   Должно показать что-то вроде: `git version 2.xx.x`

## Способ 2: Через Microsoft Store

1. Откройте **Microsoft Store**
2. Найдите **"Git"** или **"Git for Windows"**
3. Нажмите **"Установить"**
4. Перезапустите PowerShell

## После установки Git

### 1. Настройте Git (первый раз):

Откройте PowerShell и выполните:

```powershell
git config --global user.name "Ваше Имя"
git config --global user.email "ваш@email.com"
```

Замените на ваши реальные данные.

### 2. Установите Git LFS (опционально):

Если хотите использовать Git LFS для больших файлов:

1. Скачайте с: https://git-lfs.github.com/
2. Или после установки Git выполните:
   ```powershell
   git lfs install
   ```

### 3. Проверьте, что все работает:

```powershell
git --version
git lfs version
```

## Если что-то пошло не так

### Git не найден после установки

1. **Перезапустите PowerShell** (закройте и откройте заново)
2. Проверьте переменные окружения:
   - Откройте "Параметры системы" → "Дополнительные параметры системы"
   - "Переменные среды" → проверьте, что Git в PATH
3. Перезагрузите компьютер (в крайнем случае)

### Альтернатива: Без Git LFS

Если не хотите устанавливать Git LFS, можно работать без него:
- См. `QUICK_START_NO_LFS.md`
- См. `ALTERNATIVE_WITHOUT_LFS.md`

## Где вводить команды?

### В PowerShell:

1. Нажмите **Win + X**
2. Выберите **"Windows PowerShell"** или **"Терминал"**
3. В открывшемся окне введите команды

### Или в терминале редактора:

- **VS Code / Cursor**: Нажмите `Ctrl + ~` (тильда) для открытия терминала
- Убедитесь, что выбран **PowerShell** (внизу справа)
